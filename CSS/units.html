Relative units in CSS

Relative units in CSS are used to specify sizes and lengths in relation to other elements or the viewport. They allow for flexible and responsive designs that can adapt to different screen sizes and user preferences. Some common relative units include:

  • em: The 'em' unit is relative to the font size of the parent element. For example, if the parent element has a font size of 16px, then 1em would be equal to 16px.
  • rem: The 'rem' unit is relative to the root element's font size. It is similar to 'em', but it always refers to the font size of the root element (usually the element).
  • vw: The 'vw' unit is relative to the width of the viewport. 1vw is equal to 1% of the viewport's width.
  • vh: The 'vh' unit is relative to the height of the viewport. 1vh is equal to 1% of the viewport's height.
  • %: The percentage unit is relative to the parent element's size. For example, if a div has a width of 200px and you set its width to 50%, it will be 100px wide.
